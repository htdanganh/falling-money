This program is a game that collects falling cash. The player must race an opponent to reach 25 points. The first player to reach 25 points wins. 

The game includes a start, instruction, play, and end screen. The user must click the 'enter' button to initiate the game.

To move Player One (Brown), use The arrow keys. 

The end screen displays who won first. 

The player can restart the game at any point by hitting the select (backspace) key.
